Empty Flash Message and Unknown Agent Type Error when submit work with DOI and no publisher.
Descriptive summary When creating a work and choosing to mint a DOI and then manually emptying the publisher field, an empty red flash message appears.  Then when I fill out the Publisher field and try to create a work, we get an unknown agent type error. Expected behavior It is expected that the flash message should have some content and that the work should submit after filling in the publisher. Actual behavior   Steps to reproduce the behavior  Creat Work Mint DOI Go back to Metadata form. Remove Publisher. Submit Work. Fill in Publisher. Select Mint DOI. Observe Error
