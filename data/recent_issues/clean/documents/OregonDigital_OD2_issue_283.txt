Easy deployment for OD2 
Descriptive summary Tie together the pieces of infrastructure to provide the ability for hubot deployment of OD2. Expected behavior System option   Request a beta invite to Github Actions to implement deployment there  Stand up an instance of Jenkins X (or plain Jenkins?) and implement the deployment and configurations there  Script option   Clone OD2-infra project to an appropriate server, this must have kubectl, docker, and all ruby dependencies to build the OD2 project  Create a build and deployment script that automates the ansible build and k8s deployment process on the server. See #282  osulp/hubot : add configuration to apps.json and merge  (prod server) deploy app : add shared/config/apps/od2.yml with configurations and ssh command line to execute the build and deployment script
