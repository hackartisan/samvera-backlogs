Identify criteria for which file to generate a thumbnail from or see in the Viewer
Prioritize the following File Uses for displaying to the Viewer:  If a Service File has been generated automatically as part of a bulk ingest (may be stored separately from Curate preservation storage context/may not be associated with a Curate FileSet), use that Service File for IIIF delivery If a Service File has been supplied and assigned to a Curate FileSet, use that file If an Intermediate File is assigned, use that file If only a Preservation Master File is assigned, use that file  For generating Curate FileSet thumbnails, use the following:  If an Intermediate File is assigned, use that file If only a Preservation Master File is assigned, use that file
