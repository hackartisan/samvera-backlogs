Collection Edit page should include Configure Facets tab
Descriptive summary IF Configurable Facets is selected for this collection type, the collection edit screen should include a new tab labelled "Configure Facets". This tab will take the user to a screen that allows for the following abilities:   turn facets on or off for this collection  change the order of the facets for this collection  edit the label of a facet for this collection  The above configuration of facets in a collection should only appear for this collection. Related work #527 Accessibility Concerns  navigation and selection of elements on the page
