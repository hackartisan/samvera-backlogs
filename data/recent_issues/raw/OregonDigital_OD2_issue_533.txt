Create a new option for Collections which enables configurable facets
Descriptive summary When creating a new collection type, there are multiple options that you can enable in which "append" functionality to the collection type. There needs to be a choice in the settings tab after you create the new collection type that enables that collection type to have configurable facets. The workflow would be as follows:   Go to collection type tab  Create new collection type  Immediately after creation, go to settings tab  Click the check box to enable configurable faceting  Save the changes  By following these steps, when you create a collection of the previously created work type, it should allow you to configure the facets for that collection. This includes being able to order the facets, change the label, and enable/disable certain facets. This data will need to be tracked per collection and should be configurable per collection.
