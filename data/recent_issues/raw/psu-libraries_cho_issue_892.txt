Support for Audio and Video Bags/Work Types
In order to be able to ingest audio and video resources, we need to be able to support:  Resource-level administrative files [#1054] _media use type suffix _caption use type suffix  _media files are images of the original media and/or their housing. These files should be restricted to collection editors, metadata managers, and repository managers. _caption files are time-coded captions in a discrete files. This same information may also be embedded in other media containers. _media and _caption may need to be considered a new file set type. Not currently planned to be included in the CSV.
