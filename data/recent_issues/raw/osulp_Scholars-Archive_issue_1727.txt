Torrent Problems Report Form
Descriptive summary When we have BitTorrent issues related to dataset download we usually need several pieces of information to resolve the problem (error messages, screenshots, torrent files). Instead of going back and forth via e-mail we could have a Torrent Problem report form that asks for all the information we need. Expected behavior Existence of Torrent Problem report that asks for the following information:  Description of the problem Torrent file used (attached to the form) Error messages Screenshots (if needed) E-mail from the user so that we can get back  Actual behavior There is no form.
