Adds benchmarking rake tasks and updates Travis config
Run all the testing benchmarks with the defaults rake donut_testing:all or optionally pass in a length parameter to change the size of the benchmark such as rake donut_testing:all[10000]. The default is 10 iterations for each benchmark. Corresponding .csv files are added to the tmp directory such as donut_collections_10.csv, donut_nested_works_10.csv, and donut_files_10.csv (the number appended to the filename is dynamic based on the length parameter of the test run). Benchmarks can be run individually, for example rake donut_testing:collections will run only the :collections benchmark.
