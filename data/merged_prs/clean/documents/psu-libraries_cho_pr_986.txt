Enforce authenticated access
Description Defines access levels, previously known as visibility, on collections, and creates a central vocabulary and transaction for assigning them. Using Blacklight's access controls gem, these levels are used to restrict access to collections, works, and file sets based on whether or not a user is authenticated to Penn State or not.
