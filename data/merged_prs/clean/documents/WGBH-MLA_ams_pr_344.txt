Changes of how we ensure Sidekiq is running
Changes AMS::ToggleSidekiq to AMS::SidekiqProcessManager to be more accurately descriptive. Adds/Modifes methods #turn_on, #turn_off, ensure_on of AMS::SidekiqProcessManager. Uses AMS::SidekiqProcessManager in new ruby script bin/scripts/ensure_sidekiq_running.rb to ensure the correct number of Sidekiq processes are running. Modifies deployment script bin/scripts/install.sh to use new ruby script. Changes concurrency from 10 to 1 in Sidekiq config file. Changes rake task verify_service:sidekiq to use new AMS::SidekiqProcessManager class.
