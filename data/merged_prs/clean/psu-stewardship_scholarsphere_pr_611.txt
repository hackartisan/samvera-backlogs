Show representatives with a NullPresenter that uses a default image
This creates the concept of a null presenter that is used as a representative for a work when one is not present. There are many reasons why a work might not have a representative:  a work without a file uploading in process representative is nil or just didn't get set due to an error  This also shrinks the size of the default file set image which is used for the null presenter or any file set that isn't an image or file type with a specific icon. Currently, works without representatives or missing representatives would look like:   This PR will make them all uniform in appearance:   Either the representative will have a download link, such as a dataset, or it will not. The latter would be in the case of a bad work, work without a file, or a work that is in process.
