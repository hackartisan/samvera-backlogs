[Security] Bump devise from 4.6.1 to 4.7.1
Bumps devise from 4.6.1 to 4.7.1. This update includes a security fix.  Vulnerabilities fixed Sourced from The Ruby Advisory Database.  Devise Gem for Ruby confirmation token validation with a blank string Devise before 4.7.1 confirms accounts upon receiving a request with a blank confirmation_token, if a database record has a blank value in the confirmation_token column. However, there is no scenario within Devise itself in which such database records would exist.
