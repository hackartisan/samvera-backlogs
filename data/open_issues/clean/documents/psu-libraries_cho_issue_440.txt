Filenames for User Downloads
As an archivist, I would like the original filename to be used for user downloads and exports, So that file naming conventions are maintained and I can reverse-lookup the content if necessary. Nathan T. As an art history curator, I would like the work title to be used as a filename for user downloads, So that my students have a human-readable filename because my filenames are opaque. The original filename should be indexed, however, as well as my local record id as an alternate identifier. Catherine A. The default CHO behavior should be the first user story, the second would be the exception. This might be handled via a collection-property or perhaps a custom work type with processing rules for content that should behave this way.
