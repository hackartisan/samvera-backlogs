Dashboard User Show Page
STORY As a registered user I should be able to access my own user show page.  As a registered user with at least one admin role I should be able to access the user show pages of users that play a role on the presses I administer. ACCEPTANCE   There is a privileged user show page restricted to the user.  There is a privileged user show page restricted to admins of presses the user plays a role and the content is filtered by the presses the admin administers.  The content is NOT filtered if viewed by a platform admin.
